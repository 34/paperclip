# 这是什么（What is it）

Paperclip 是一个 Node 服务与 UI，用于编排一队 agents 来运营业务。

自带 agents、分配目标，并追踪 agent 的工作与成本。

看起来像任务管理器，但具备专门能力与 agent 协同。

**高品味、好用**

# Paperclip 适合你，如果：（Paperclip is right for you if）

- 你想打造自治公司
- 你想协调多种 agents（OpenClaw、Codex、Claude、Cursor 等）为共同目标协作
- 你开了 20 个 Claude Code 终端（且搞不清谁在做什么）
- 你想让 agents 7×24 自主运行，但仍能审计工作并在需要时介入
- 你想监控成本
- 你想要一套像任务管理器一样管理 agents 的流程
- 你想用手机管理自治业务

# 不是什么（What is not）

Paperclip 不是：

- 代码审查工具
- 限定你必须用哪种 agent
