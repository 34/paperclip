# Investigation: 获客渠道分析 — 小红书 vs 闲鱼

**Date:** 2026-04-10
**Branch:** learn
**Status:** DONE

---

## 核心结论

两个渠道都要做，但优先级和策略不同。小红书是主阵地（内容获客 + 付费投流），闲鱼是补充（低价走量 + 即时成交）。都不依赖微信自动化。

---

## 渠道对比

| 维度 | 小红书 | 闲鱼 |
|------|--------|------|
| **定位** | 内容种草 → 长期品牌 | 低价走量 → 即时成交 |
| **获客成本** | 5-30元/咨询 (薯条75元起) | 10元/天推广起步 |
| **客单价上限** | 300-2000元 (中高端) | 20-100元 (低端) |
| **付费投流** | 薯条(75元起) + 聚光(5000元起) | 闲鱼币推广 + 付费推广(10元/天) |
| **竞争密度** | 中端激烈但可差异化 | 饱和，价格战严重 |
| **合规风险** | 低 (正规内容平台) | 中 (虚拟服务属灰色地带) |
| **需要企业资质** | 聚光需要, 薯条不需要 | 不需要 |
| **交付方式** | 引导到企微/私域成交 | 平台内支付宝担保交易 |
| **季节性** | 强 (5月答辩, 12月年终) | 强 (同上) |
| **净利润率** | 80%+ | 80%+ (但客单低) |

## 小红书策略

### 内容打法 (零成本启动)

**5种有效内容类型:**

| 类型 | 目的 | 怎么做 |
|------|------|--------|
| **前后对比** | 爆款触达, 最有效 | 客户丑PPT vs 重新设计的PPT, 视觉冲击 |
| 痛点+解决方案 | 情感共鸣 | "答辩PPT避坑指南", "导师最讨厌的10个PPT错误" |
| 干货教程 | 搜索流量, 建立权威 | 答辩PPT排版技巧, 配色方案推荐 |
| 客户评价截图 | 信任/转化 | 聊天记录, 客户好评, 交付案例 |
| 模板预览 | 引流 | "免费答辩PPT模板" — 高互动但低意图 |

**标题公式:** 前18字包含2个核心关键词
- "毕业答辩PPT代做 前后对比 导师直夸好看"
- "答辩PPT避坑 这5个错误99%的人都在犯"

**算法关键:** 评论、分享、关注的权重 > 点赞和收藏。帖子要引导互动: "你会选哪个版本？"

### 付费投流路径

```
阶段1 (第1个月): 个人号 + 免费内容
  → 0元, 发布10-15篇笔记, 积累搜索权重

阶段2 (第2个月): 薯条测试
  → 300-500元测试预算
  → 对表现好的笔记投薯条(75元/次)
  → 测试哪种内容产生私信咨询

阶段3 (第3个月+): 注册个体工商户 → 专业号 → 聚光
  → 600元/年认证费
  → 聚光充值5000元起(通过代理)
  → 关键词搜索广告: "毕业答辩PPT" "PPT代做"
  → 目标CPC: 0.8-3元/点击
```

### 转化路径

小红书**禁止直接放微信号** (7天封禁)。合规转化方式:

**路径A: 群聊 → 企微 (推荐)**
```
笔记 → "群聊: 答辩PPT模板领取" → 群内分享 → 引导到企业微信 → 成交
```

**路径B: 私信通 (专业号)**
```
笔记 → 用户点"咨询" → 私信中分享企微联系方式 → 成交
```

**路径C: 小红书店铺 (1000粉丝后)**
```
笔记 → 直接挂虚拟商品链接 → 平台内购买
```

推荐: **A+B组合**, 用企微承接客户, 不依赖个人微信。

## 闲鱼策略

### 关键发现: 虚拟服务是灰色地带

闲鱼官方禁止"虚拟服务"上架, 但对小额违规容忍度较高。实操要点:

- **不能写"代做"或"代写"** — 用"PPT设计"、"PPT美化"、"PPT定制"
- 商品可能被删, 账号可能被封 — **闲鱼不能做唯一渠道**
- 需要多账号分散风险

### 定价策略

闲鱼价格远低于行业平均:

| 层级 | 价格 | 内容 | 定位 |
|------|------|------|------|
| 引流款 | 9.9-19.9元 | 模板包, 自动发货 | 拿好评, 涨信用 |
| 主力款 | 20-50元 | 定制毕业答辩PPT, 简单设计 | 走量 |
| 利润款 | 50-100元 | 定制设计 + 修改 | 筛选优质客户 |

**对比小红书客单价:** 闲鱼50元 vs 小红书500元。差10倍。闲鱼适合走量练手和建立好评, 不适合赚钱。

### 交付方式

1. 买家下单 → 聊天确认需求 → 制作 → **网盘链接发文件** → 买家确认收货
2. 支付宝担保交易, 安全但需买家手动确认
3. 48小时内必须响应退款请求

### 推广

- 闲鱼币推广: 免费(可赚闲鱼币), 曝光计费
- 付费推广: **10元/天起步**, ROI通常1:3到1:80
- "擦亮"功能: 每天5-7点刷新商品, 提升排名

## 推荐执行方案

### 渠道优先级

```
主阵地: 小红书 (内容+投流, 高客单, 建品牌)
补充: 闲鱼 (低价走量, 练手+好评, 风险分散)
承接: 企业微信客服 (成交+交付, 合规)
```

### 第1周启动计划

| 天 | 小红书 | 闲鱼 |
|----|--------|------|
| Day 1 | 注册个人号, 发布3篇笔记 (前后对比×2 + 干货×1) | 注册账号, 发布5个商品 (不同关键词) |
| Day 2 | 发布3篇笔记 (痛点×1 + 模板预览×1 + 评价×1) | 发布5个商品, "擦亮"所有商品 |
| Day 3 | 发布3篇笔记, 回复所有评论 | 擦亮商品, 回复咨询 |
| Day 4 | 分析哪篇数据最好, 追加同类型内容 | 分析浏览量, 调整标题/价格 |
| Day 5 | 对表现最好的笔记投薯条(75元) | 花10元测试付费推广 |
| Day 6 | 收集咨询, 通过群聊引导到企微 | 处理第一个订单 |
| Day 7 | 复盘数据, 制定第2周内容计划 | 复盘数据, 调整策略 |

| 项目 | 费用 | 说明 |

## 验证指标 (4周内)

| 指标 | 小红书目标 | 闲鱼目标 |
|------|-----------|---------|
| 发布内容 | 30+篇笔记 | 15-20个商品 |
| 曝光 | 10,000+ | 5,000+ |
| 咨询 | 20+条私信 | 10+条 |
| 成交 | 5+单 | 5+单 |
| 收入 | 2,500+元 (500元/单) | 250+元 (50元/单) |

如果小红书4周内能产生20+咨询和5+成交, 验证通过, 值得加大投入。

闲鱼主要价值不在收入, 而在: (1) 练手AI生成流程 (2) 积累好评案例 (3) 测试定价。

## 对架构的影响

1. **wechat-plugin → 重新定义为 channel-plugin**
   - 不再绑定微信, 而是统一的消息入口
   - 小红书群聊消息 + 闲鱼聊天消息 → 统一进入 Paperclip Issue 系统
   - 企业微信客服作为主要成交渠道

2. **获客不依赖任何单一平台的自动化**
   - 小红书: 内容运营 (人工发布) + 薯条/聚光 (付费)
   - 闲鱼: 商品运营 (人工上架) + 付费推广
   - 企微: 客服承接 (AI Agent 辅助)

3. **Paperclip Agent 职责调整**
   - 客服专员: 处理企微客服消息 (不再是微信个人号)
   - 新增: 营销Agent — 生成小红书/闲鱼投放素材
   - CEO Agent: 监控两个渠道的ROI, 调整预算分配
