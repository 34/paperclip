## 核心决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 切入点 | 毕业答辩PPT | 4-6月旺季, 结构标准, 客单价适中 |
| 路径 | B→A渐进 (人工→自动化) | Tome融$81M倒闭, 先验证再扩展 |
| 获客渠道 | 小红书(主)+闲鱼(辅) | 微信自动化已死, 小红书ROI验证 |
| 客户沟通 | 企业微信客服API | 合规, 免费, 支持文件传输 |
| PPT生成 | python-pptx+模板填充 | 质量8-9/10, 成本最低 |
| 运营架构 | Paperclip虚拟公司 | Agent团队自主运营, Issue管理订单 |
