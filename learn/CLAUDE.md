# learn/ 目录规范

本目录用于 Paperclip 项目的学习、探索和实验记录。

## 命名规则

- **文件名和目录名优先使用中文**。例如: `ppt生成技术调研.md` 而非 `ppt-generation-research.md`
- 例外: 代码文件、配置文件、必须用英文的文件名 (如 CLAUDE.md, README.md, package.json) 保持英文
- 每个 Markdown 文档的标题 (一级标题) 也使用中文

## 内容规范

- 文档内容使用中文撰写, 技术术语可保留英文 (如 API、Agent、Plugin)
- 每个实验项目目录包含 README.md 作为入口, 说明背景、目录结构、阅读顺序
- 调研类文档需注明日期和结论摘要, 放在文件开头
- 实验记录需包含: 目标、过程、结果、决策, 便于后续回顾
