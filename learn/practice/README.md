# 实践操作记录

完整的端到端使用流程记录。偏向"怎么做"的操作手册，可复现。

## 建议的实践方向

### 搭建流程
- 从零搭建第一个 AI 公司（完整流程）
- 搭建多层组织架构（CEO → CTO → 工程师）
- 配置多项目管理流程

### 自动化
- 配置 Routine 实现自动化工作流
- 设置 Webhook 触发外部集成
- 配置 Routine 变量和模板

### 治理
- 设置公司级和 Agent 级预算
- 配置审批策略和执行策略
- 建立成本监控体系

### 开发
- 开发自定义插件（从 hello-world 到实用插件）
- 开发自定义适配器（接入新的 Agent 运行时）
- 修改核心功能（二次开发实战）
