# Paperclip 学习与实践

基于 Paperclip 项目的深度学习、探索和实践记录。

## 目录结构

```
learn/
├── README.md              ← 你在这里
│
├── analysis/              项目深度分析
│   ├── README.md
│   └── 01~09 分析报告
│
├── exploration/           功能探索与能力验证
│   └── README.md
│
├── practice/              实践操作记录
│   └── README.md
│
├── experiments/           公司搭建实验
│   └── README.md
│
└── notes/                 零散笔记与备忘
    └── README.md
```

## 各目录说明

### `analysis/` — 项目深度分析
已完成的项目源码级分析，包括架构、数据模型、核心业务逻辑、API 接口等。适合作为二次开发的参考手册。

### `exploration/` — 功能探索与能力验证
针对 Paperclip 各功能模块的动手探索。比如：心跳机制怎么触发、Routine 的 Cron 怎么配、插件怎么装、适配器怎么切换等。每个文档聚焦一个具体功能点。

### `practice/` — 实践操作记录
完整的使用流程记录。比如：从零搭建一个 AI 公司、配置一条自动化流水线、设置预算管控策略等。偏向"怎么做"的操作手册。

### `experiments/` — 公司搭建实验
不同方向、不同架构的 AI 虚拟公司搭建尝试。比如：软件开发公司、内容创作公司、数据分析公司等。记录设计思路、搭建过程、运行效果、经验教训。

### `notes/` — 零散笔记与备忘
临时想法、踩坑记录、待办事项、灵感备忘等。不要求格式，快速记录。
